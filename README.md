# Shashank Portfolio

Modern single-page portfolio website for Shashank Kallahalli Suresh, redesigned with a clean pastel Designfolio-inspired visual style.

## Live Content

- Hero with browser-style intro card
- Modular toolkit/services cards
- Experience cards with quantified impact
- Separate project sections:
  - Product & App Projects
  - Analytics & Research Projects
- Featured BuildFolio project
- Publications and contact sections

## Tech

- HTML5
- CSS3 (custom responsive design)
- Vanilla JavaScript
- Font Awesome icons
- Google Fonts

## Local Preview

From the project root:

```bash
python3 -m http.server 8000
```

Then open:

`http://localhost:8000/index.html`
